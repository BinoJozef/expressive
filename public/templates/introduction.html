Introduction Expressive

Expressive is bundled web app call M-EAN (MySQL, Expressjs, Angularjs, and Nodejs) it also integrated with Angular Material as a theme.

Feature
  • Build System Gulp
  • Package manager Bower
  • Framework Angularjs
  • Theme Angular Material
Installation
  1. Install gulp-cli and bower globally
  2. Clone expressive 
    $ git clone https://github.com/asrul10/expressive.git
  3. Install node packages 
    $ npm install
  4. Install bower packages 
    $ bower install
  5. Config databses in config/index.js
  6. Sync models 
    $ node sync.js
  7. Run expressive in development 
    $ gulp build 
    $ gulp serve
  8. For production 
    $ gulp production
